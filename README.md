# Structure du Projet TTO-EXTENSION

```
TTO-EXTENSION/
├── src/
│   ├── js/
│   │   ├── content.js                 # Point d'entrée principal
│   │   ├── modules/
│   │   │   ├── sidebar.js            # Gestion de la barre latérale
│   │   │   ├── eventHandlers.js      # Gestion des événements
│   │   │   └── downloadManager.js    # Gestion des téléchargements
│   │   └── utils/
│   │       └── constants.js          # Constantes globales
│   ├── css/
│   │   └── styles.css                # Styles de l'extension
│   └── assets/
│       └── icons/                    # Icônes de l'extension
├── manifest.json                      # Configuration de l'extension
└── README.md                         # Documentation du projet
```
